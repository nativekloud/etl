# ETL for CLojure

A Clojure ETL based on Singer open source standard [docs](https://github.com/singer-io/getting-started)

## Usage

``` shell
java -jar target/etl-0.1.0-SNAPSHOT-standalone.jar --config resources/tap-csv-config.json tap --type csv
```
